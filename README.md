#Expense Tracker (Трекер расходов)
Описание программы
Expense Tracker - это простое GUI-приложение, разработанное для отслеживания личных расходов. Оно позволяет пользователю вводить, сохранять и анализировать свои финансовые траты, предоставляя возможность фильтрации данных по категории и дате. Приложение предоставляет удобный интерфейс для легкого отслеживания и управления личным бюджетом.

Основные функции
Добавление расхода: Возможность ввести сумму, выбрать категорию (еда, транспорт, развлечения) и указать дату.
Фиксация данных: Все введенные данные хранятся в формате JSON, что обеспечивает сохранность и удобность в дальнейшем использовании.
Подсчет расходов: Программа может подсчитывать общую сумму расходов за выбранные даты.
Фильтрация данных: Пользователь может фильтровать расходы по категории и дате, что упрощает анализ финансов.
Как использовать
Запустите программу, используя Python 3.
Введите сумму ваших расходов в соответствующее поле.
Выберите категорию расхода из выпадающего списка.
Введите дату в формате YYYY-MM-DD.
Нажмите кнопку "Добавить расход", чтобы сохранить запись.
Все расходы отображаются в списке ниже.
Примеры использования
Отслеживание ежедневных расходов на еду и развлечения.
Анализ финансов за определенный период (например, за месяц).
Подсчет общей суммы расходов для планирования бюджета.
